*Pokédex*

Uma Pokédex interativa desenvolvida com HTML, CSS e JavaScript utilizando a PokéAPI para buscar informações de Pokémons em tempo real.

*Funcionalidades*

- Pesquisa de Pokémon por nome  
- Integração com a PokéAPI  
- Exibição de imagem do Pokémon  
- Exibição de ID  
- Exibição de tipo  
- Exibição de altura  
- Exibição de peso  
- Badge dinâmica com cor baseada no tipo do Pokémon  
- Layout responsivo  
- Fundo animado com Pokébolas  
- Tratamento de erros para Pokémon inexistentes  

*Tecnologias Utilizadas*

- HTML5
- CSS3
- JavaScript
- PokéAPI

*API Utilizada*

O projeto utiliza a API pública:

- https://pokeapi.co

Exemplo de requisição:

    fetch(`https://pokeapi.co/api/v2/pokemon/pikachu`)

*Interface*

O projeto possui:

- Landing Page inicial
- Fundo animado
- Design inspirado em Pokédex
- Cards dinâmicos
- Tipagem colorida dos Pokémons

*Tratamento de Erros*

Caso o Pokémon pesquisado não exista:

- Uma imagem padrão é exibida
- Os campos são resetados
- O sistema informa que o Pokémon não foi encontrado

*Responsividade*

A aplicação foi adaptada para funcionar em:

- Desktop
- Tablets
- Smartphones
